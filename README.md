# 微信小程序离线文档

微信小程序官方文档的离线版，Docset格式，可添加到下列软件中使用：

*Mac:* [Dash](https://kapeli.com/dash)

*Windows:* [Velocity](http://velocity.silverlakesoftware.com)

*Linux:* [Zeal](https://zealdocs.org/)
